# Reset the curse of time - level 4
Este projeto é um dois leveis de um RPG de texto em Java que ensina laços de repetição durante a sua jornada.
O objetivo do player é encontrar o artefato chamado Laço do Infinito, que permite viajar para o passado ou futuro e realizar objetivos definidos por ele.

Na minha fase, trabalhei com o laço do-while e implementei funcionalidades que tornam a experiência mais dinâmica:

- Inimigos com ASCII art, inspirados em algumas das minhas obras favoritas, aparecendo de forma aleatória durante a run, cada um com um estoque de perguntas exibidas também de forma aleatória.

- Baús aleatórios, que podem conter recompensas ou surpresas.

- Dicas ocasionais, que ajudam o player a avançar no labirinto.

- Boss final com questões mais complexas sobre o laço do-while, reforçando o aprendizado antes de seguir para o próximo nível.

Esses atributos de aleatoriedade fazem com que exista a possibilidade de milhares de runs diferentes.
